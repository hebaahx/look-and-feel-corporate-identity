# Titel
SnappThis, dit is een webapplicatie waarin je foto's in een snappmap kan zetten. 

## Beschrijving
SnappThis is een project waarin we werken aan de look & feel en styleguide van de applicatie. We ontwerpen kleuren, typografie, UI-elementen en de algemene stijl zodat de app er duidelijk en consistent uitziet. Deze fase draait vooral om design, consistentie en gebruikerservaring, zodat SnappThis er professioneel uitziet en prettig werkt.

Hier is een link naar de figma waar de desgin in staat: https://www.figma.com/design/lphsL70W8JfHtddUdaon3k/2025snappthisDesign?m=auto&t=PYT30zV6UCJj3Wk0-6

Hier is een live link van de page in mijn github: https://hebaahx.github.io/look-and-feel-corporate-identity/

## Kenmerken
In dit project werken we met een gezamenlijke styleguide, die ik samen met mijn projectgroep heb ontwikkeld. Deze styleguide gebruiken we in onze eigen repositories, zodat alle onderdelen dezelfde look & feel hebben.

HTML
Duidelijke, semantische structuur (zoals header, main, section, nav, footer).
Herbruikbare componenten zoals buttons en navigatieblokken.
Consistente namen, gebaseerd op de styleguide.

CSS
Kleuren, typografie en spacing komen uit de gezamenlijke styleguide.
We gebruiken CSS-variabelen (bijv. --primary-color) voor een consistente stijl. Focus op responsive design.

Styleguide
Gemaakt in samenwerking met het projectteam.
Bevat kleurenpalet, typografie, UI-componenten en spacing-regels.
Iedereen kan deze in hun eigen project gebruiken voor een uniforme stijl.
